## ✨ About

**BinDawood Dulha House** is a single-page, fully responsive luxury website built for a heritage groom-wear boutique based in **Dara Arraian, Sialkot**. The site showcases custom-tailored **sherwanis, prince coats, pent coats, waist coats, shalwar qameez, stone-embroidered suits, and groom turbans (kula)** — blending royal Pakistani wedding tradition with a modern, elegant digital storefront.

Every interaction — browsing, booking, or inquiring — routes seamlessly into **WhatsApp**, keeping the experience personal and conversion-friendly for a market that lives on mobile chat.

> *"Crafting the perfect dulha since generations."*

## 🎯 Key Features

| Feature | Description |
|---|---|
| 🖼️ **Dynamic Product Galleries** | Custom-built image sliders for multi-photo products (1, 2, 3+ images handled automatically) |
| 💬 **WhatsApp-First Conversion** | Every "Inquire" button opens a pre-filled WhatsApp message — zero friction |
| 📅 **Smart Appointment Booking** | Full form with service type, date, time & wedding date — submits straight to WhatsApp |
| 🗂️ **Category Filtering** | Instant client-side filtering across 8 product categories |
| 🌟 **Single Page Application** | Smooth in-page routing between Home, Collections, Tailoring, About, Gallery, FAQ & Contact — no reloads |
| 📱 **Fully Responsive** | Optimized layouts for mobile, tablet & desktop |
| 🎬 **Scroll Reveal Animations** | Elegant fade-and-rise transitions powered by Intersection Observer |
| ⭐ **Testimonial Carousel** | Auto-rotating client reviews with manual navigation dots |
| ❓ **Accordion FAQ** | Clean, expandable answers to common client questions |
| 🔍 **SEO + Schema Optimized** | Open Graph tags, Twitter Cards & `ClothingStore` JSON-LD structured data baked in |

## 🛠️ Tech Stack

<div align="center">

| Layer | Technology |
|---|---|
| **Markup** | Semantic HTML5 (single-file SPA) |
| **Styling** | Custom CSS3 — CSS Variables, Grid, Flexbox |
| **Typography** | [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) (display) + [Josefin Sans](https://fonts.google.com/specimen/Josefin+Sans) (body), via Google Fonts |
| **Interactivity** | Vanilla JavaScript — no frameworks, no build step |
| **Animations** | Intersection Observer API for scroll reveals |
| **Integration** | `wa.me` deep links for WhatsApp-based booking & inquiries |

</div>

**Design language:** Deep emerald green, jet black & antique gold — evoking royal Mughal-era textiles, finished with a fine `jali` lattice pattern motif throughout.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Abd-Abdullah83/BinDawood.git

# Open directly in a browser — no build step required
cd BinDawood
open index.html      # or just double-click it
```

No dependencies, no `npm install`, no compiling. It's a single static HTML file — deploy it anywhere (GitHub Pages, Netlify, Vercel, or any shared host).
